# biblioteca-de-babel
Sistema de gestión bibliotecaria

## Instalación
Clonar el repositorio:

```bash
git clone https://github.com/JulianBodo/biblioteca-de-babel.git
cd biblioteca-de-babel
```

Instalar dependencias:

```bash
npm install
```

Configurar variables de entorno:

```bash
cp .env.example .env
```

Levantar PostgreSQL:

```bash
docker compose up -d
```
